Harkányi Gergely

Egy olyan weboldal, amely túra útvonalakat ajánl egy galéria és a túrák rövid leírása mentén. A weboldal tartalmaz még egy számológépet, amely segít az utazás további szempontok menti tervezésében

JavaScript helye és működése
/js/form-validation.js — a kapcsolat űrlap validációjáért felel (név, email, résztvevők, dátum, túratípus ellenőrzése).
/js/kalkulator.js — az utazásköltség-kalkulátor logikája; kiszámolja a szállás, utazás és belépők költségeit, és localStorage-ba mentési lehetőséget biztosít.
/js/main.js — általános UI funkciók (mobil menü, galéria lightbox).
